# VSCode Tan Extension

A Visual Studio Code (VSCode) extension for the Tan language.

## Setup

```sh
npm i
```

## Usage

Package as VSIX:

```sh
vsce package
```

or

```sh
npm run package
```

To install, right-click from VSCode sidebar and select `Install Extension VSIX`.
